# 🏡 MERN Cottage Rent

**A full-stack rental web app for listing and renting imaginary cottages.**  
Create an account, explore homes, and simulate a cozy getaway!

![Node.js](https://img.shields.io/badge/Node.js-v20.11.0-green)
![Express](https://img.shields.io/badge/Express-v4.18.2-orange)
![React](https://img.shields.io/badge/React-v18.2.56-blue)

---

## 📌 Features
- 🔐 User Authentication (Sign up / Log in)
- 🏠 Browse cottages and rental listings
- 💾 Save rentals to profile
- 🛠 Update account details

## ⚙️ How to Run Locally

```bash
# Step-by-step instructions
1. Download and extract the project files
2. Navigate to the root folder:
   cd MERN-Cottage-Rent-main

3. Install dependencies:
   npm run install-deps

4. Build frontend production files:
   npm run build-project

5. Add sample cottage listings:
   add-data

6. Start production build:
   npm run start-prod

7. Open your browser and go to:
   http://localhost:8000
```
